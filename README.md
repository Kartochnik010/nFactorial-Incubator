## Как запустить?

##### Скачиваем:
```
git clone github.com/Kartochnik010/nFactorial-Incubator
```

##### Запускаем:
```
cd nFactorial-Incubator
python3 server.py
```

##### Если у вас еще нет всех необходимых зависимостей то устаноновите их: 
```
pip install <имя необходимой зависимости>
```
